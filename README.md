# Social Network Analysis and Recommendation System

A beginner-friendly Python project that analyzes a social network dataset and provides recommendations based on user connections and interests.

## Features

- Implemented using Python and Jupyter Notebook
- JSON-based dataset processing
- People You May Know recommendation
- Pages You Might Like recommendation

## Project Files

- `raw_data.json` – Raw social network dataset
- `cleaned_data.json` – Cleaned dataset after preprocessing
- `data_loading.ipynb` – Loads and explores the dataset
- `data_cleaning.ipynb` – Cleans and preprocesses the data
- `people_you_may_know.ipynb` – Recommends users based on mutual connections
- `pages_you_may_like.ipynb` – Recommends pages based on user interests

## Technologies Used

- Python
- Jupyter Notebook
- JSON
